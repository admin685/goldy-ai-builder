## Part 1 — Deployment Fix (CRITICAL, do first)

### 1.1 Check why server fails to start

Run these checks:
```bash

# Check package.json start scripts
cat package.json
cat artifacts/api-server/package.json

# Check .replit deployment config
cat .replit
```

### 1.2 Fix port configuration

The server MUST listen on the PORT environment variable that Replit provides:
```javascript
// In app.ts — ensure this is correct:
const PORT = parseInt(process.env.PORT || '8080', 10)
app.listen(PORT, '0.0.0.0', () => {
  console.log(`Server running on port ${PORT}`)
})
```

`0.0.0.0` is critical — without it Replit can't route traffic to the server.

### 1.3 Fix .replit deployment config

Ensure `.replit` has:
```toml
[deployment]
run = ["pnpm", "--filter", "@workspace/api-server", "run", "start"]
build = ["pnpm", "--filter", "@workspace/api-server", "run", "build"]
```

### 1.4 Rebuild and test

```bash
cd artifacts/api-server && pnpm run build
# Confirm dist/index.cjs exists and is not empty
ls -la dist/
```

### 2.3 Build Pipeline Check

Test a minimal build end-to-end:
```bash
# Get auth token
TOKEN=$(curl -s -X POST $BASE/api/auth/login \
  -H "Content-Type: application/json" \
  -d "{\"email\":\"$ADMIN_EMAIL\",\"password\":\"$ADMIN_PASSWORD\"}" \
  | node -e "const d=require('fs').readFileSync('/dev/stdin','utf8'); console.log(JSON.parse(d).token)")

# Reset build state first
curl -s -X POST $BASE/api/admin/reset-build \
  -H "Authorization: Bearer $TOKEN"

# Start a build
curl -s -X POST $BASE/api/build \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"idea":"simple hello world page"}'

# Poll status for 60 seconds
for i in {1..12}; do
  sleep 5
  STATUS=$(curl -s $BASE/api/status | node -e "process.stdin.resume();let d='';process.stdin.on('data',c=>d+=c);process.stdin.on('end',()=>console.log(JSON.parse(d).status))")
  echo "Status: $STATUS"
  if [ "$STATUS" = "done" ] || [ "$STATUS" = "error" ]; then break; fi
done

curl -s $BASE/api/status | node -e "process.stdin.resume();let d='';process.stdin.on('data',c=>d+=c);process.stdin.on('end',()=>{const r=JSON.parse(d);console.log('Final status:',r.status);console.log('Result URL:',r.result?.url);console.log('Error:',r.error)})"
```
